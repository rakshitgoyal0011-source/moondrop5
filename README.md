# Moodboard

Moodboard is a single-page, editorial landing experience for an AI aesthetic discovery platform. Users describe a mood or vibe, and the interface presents a living moodboard made of visuals, colors, music, fashion, quotes, and ambient inspiration.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- The site is intentionally cream-first, charcoal-forward, and lightly accented with burnt coral.
- Motion stays subtle: scroll reveals, hover shifts, and a small pointer trail in the preview section.
- The layout is responsive and collapses the asymmetrical grids into a single column on smaller screens.
